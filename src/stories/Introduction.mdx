# TutorConnect Frontend

Storybook documents shared UI components and patterns used across the TutorConnect tuition marketplace.

## Stack

- **Next.js 16** (App Router) + **React 19** + **TypeScript**
- **Tailwind CSS v4** + **shadcn/ui**
- **react-if** for declarative conditionals
- **Zod + react-hook-form** for auth validation

## Running locally

```bash
npm install

npm run dev         # http://localhost:3000
```

Browse **Common** in the sidebar for component stories with props, variants, and usage notes.
